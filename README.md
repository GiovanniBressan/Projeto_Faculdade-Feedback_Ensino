Instruções sobre o Projeto_Faculdade-Feedback_Ensino:

1) Baixar o pacote do repositorio em RAR.
2) Vai selecionar a pasta do projeto no VS Code.
3) No terminal > npm install - instalar todas as dependencias do projeto
4) Após isso, você jogará no ternimal  > cd server (que seria a pasta do servidor da API) 
5) Em seguida, no terminal escrever > node server.js - Servidor de feedback rodando em http://localhost:3001 (Back-END)
6) Ele criará um link para se rodar o servidor dos feedbacks gerados (duvida de como ver os comentarios, segue anexo video)
7) Em seguida, escreverá no terminal cd .. (para estar voltando para a pasta raiz do projeto)
8) Por fim, escreverá no terminal > npm run dev
9) E ele estará gerando o front-end da ferramenta de feedback. Tem como ver tanto nos relatorios como no link da API via JSON
